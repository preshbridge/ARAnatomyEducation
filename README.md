# AR Anatomy Education

## Project Description

AR Anatomy Education is an Augmented Reality (AR) mobile learning application developed using Unity and AR Foundation. The application is designed to improve anatomy education by allowing students to visualise and interact with a 3D model of the human heart in a real-world environment using their mobile devices.

The application uses AR plane detection to identify flat surfaces and place a realistic 3D heart model into the user's environment. Students can interact with different anatomical regions of the heart through touch-based hotspots. Selecting a heart region displays educational information about the selected anatomical structure, helping learners understand the heart's anatomy more engagingly and interactively.

The project aims to address challenges in anatomy education by providing an accessible and immersive learning experience that can be used outside traditional laboratory environments.

## Current Features

* Home Screen Interface
* AR Plane Detection
* Heart Model Placement
* Interactive Anatomical Hotspots
* Educational Information Panels
* Mobile AR Experience using ARCore
* User-Friendly Navigation
